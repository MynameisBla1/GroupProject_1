
Welcome to Quarantine Tetris 

By Sarah Toll, Y Tran and Emelia. 

This software was created to clone Tetris. This fun fulled game that can be enjoyed by 
anyone in their free time. It is useful as your everyday tetris player as while as bring 
your non-gamer to the game. 
The software itself uses a mixes of fxml and java classes. 
Java makes all the bricks and the game board and fxml is the main title screen. 

Requirement to play

You will add these two file by going to file then project structure
then in project settings you will find libraries.Then add both files to Libraries. 

lib (https://gluonhq.com/products/javafx/)
JDBC Driver (https://github.com/xerial/sqlite-jdbc/releases/tag/3.32.3.2)

You will just need these to be on your computer for them to work. 
SQLite3 (https://sqlite.org/download.html)
SQLite Browser (https://sqlitebrowser.org/dl/)
